# Discoverability checklist

To make this repo easy to find and understand on GitHub, consider configuring the following in the repository settings.

These are **not** enforced by code, but they complement the OSS scaffolding in this repo.

---

## Repository description

In GitHub → **Settings → General**:

- **Description:**  
  `StreamDeck — Multi-destination 16:9 RTMP streaming miniapp for MentraOS smart glasses.`

- **Website:**  
  Set this to your live deployment URL, for example:  
  `https://streamdeck-production.up.railway.app`

This makes the repo immediately recognizable on search and profile pages.

---

## Topics

In GitHub → **About → Topics**, add tags such as:

- `mentraos`
- `smart-glasses`
- `rtmp`
- `livestreaming`
- `bun`
- `typescript`
- `express`

These help other MentraOS and livestreaming developers discover the project.

---

## Sponsorships (optional)

If you want to accept sponsorships:

1. Fill in `.github/FUNDING.yml` with your actual handles:
   - `github: mrdulasolutions` (for GitHub Sponsors)
   - Or other platforms you use (Patreon, Ko-fi, etc.)
2. Enable the relevant integrations in your GitHub account settings

GitHub will then show a **Sponsor** button on the repo.
